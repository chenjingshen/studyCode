jQueryRegister--接口说明
验证用户名 接口地址1:_api/checkName.php
          请求方式:get
          接口参数:name 用户名

          返回数据:
            很遗憾,用不了 或 恭喜你 可以使用

--------------------------------------------------------------

验证手机号 接口地址2:_api/checkMobile.php
          请求方式:post
          接口参数:mobile 手机号

          返回数据:
            该手机号已被使用,是否找回密码 或 恭喜你,可以用(⊙o⊙)

--------------------------------------------------------------

获取验证码 接口地址3:_api/code.php
          请求方式:get
          接口参数:mobile 手机号

          返回数据:
            随机的 验证码
--------------------------------------------------------------

注册用户 接口地址4:_api/register.php
          请求方式:post
          接口参数:
            name 用户名
            mobile 电话
            pass 密码
            repass 重新输入的密码 
            code 

          返回数据:
            success
